# ✈️ Travel Budget Planner

## Objectives
- Visualize the total budget of travel through an interactive dashboard
- List travel itinerary in a presentable sheet which easy to use
- Document and count detail of each expense categories

## Tools
Google Spreadsheet

## Formulas
- Basic mathematical equations
- Data summary (SUM, SUMIF)
- Logical function (IF, IFS, SEQUENCE, IFERROR, SORT, FILTER)
- Date display (TODAY, DAYS, WEEKDAY)
- Visualization (chart, drop down, tick box, SPARKLINE)
